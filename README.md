# Calories Advisor 🍎

A simple Streamlit web app that helps users estimate calorie intake and get basic dietary suggestions using AI.

---

## 🚀 Features

* Enter food details to estimate calories
* Get AI-based suggestions
* Simple and clean user interface

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Google API (for AI responses)

---

## 📂 Project Structure

```
caloriesAdvisor/
│── app1.py
│── requirements.txt
│── .env (not uploaded to GitHub)
```

---

## ⚙️ Setup

1. Clone the repository

```
git clone https://github.com/your-username/caloriesAdvisor.git
cd caloriesAdvisor
```

2. Create virtual environment

```
python -m venv venv
```

3. Activate environment

Windows:

```
venv\Scripts\activate
```

4. Install dependencies

```
pip install -r requirements.txt
```

5. Add API key

Create a `.env` file:

```
GOOGLE_API_KEY=your_api_key
```

---

## ▶️ Run the App

```
streamlit run app1.py
```

---

## 🔐 Note

Do not upload your `.env` file or API keys to GitHub.
